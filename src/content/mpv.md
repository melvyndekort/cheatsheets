### Frequently Used
{{< table >}}
Binding | Description
--- | ---
`p` | Pause / playback mode
`f` | Toggle fullscreen
`m` | Mute / unmute audio
`v` | Subtitle visibility
`s` | Take a screenshot
`S` | Take a screenshot without subtitles
`Q` | Quit saving current position to watch later
`o` | Show progress
`O` | Toggle show progress
`T` | Toggle video window on top
{{< /table >}}

### Navigation
{{< table >}}
Binding | Description
--- | ---
`PGUP` / `PGDWN` | Next / previous chapter
`. ` / `,` | Next / previous frame
`RIGHT` / `LEFT` | Seek 5 seconds
`UP` / `DOWN` | Seek 60 seconds
{{< /table >}}

### Playback
{{< table >}}
Binding | Description
--- | ---
`Ctrl + +` | Increase audio delay
`Ctrl + -` | Decrease audio delay
`[` / `]` | Decrease / increase speed
`m` | Mute / unmute audio
`A` | Cycle aspect ratio
`1` / `2` | Decrease / increase contrast
`3` / `4` | Decrease / increase brightness
`5` / `6` | Decrease / increase gamma
`7` / `8` | Decrease / increase saturation
`9` / `0` | Decrease / increase audio volume
`w` / `e` | Zoom out / in
{{< /table >}}

### Subtitles
{{< table >}}
Binding | Description
--- | ---
`v` | Show / hide subtitles
`j` / `J` | Next / previous subtitle
`z` / `x` | Increase / decrease subtitle delay
`r` / `t` | Move subtitles up / down
{{< /table >}}

### Apple Remote
{{< table >}}
Binding | Description
--- | ---
`PLAY` | Pause / playback mode
`PLAY_HOLD` | Quit
`CENTER` | Pause
`CENTER_HOLD` | Quit
`NEXT` | Seek +10 seconds
`NEXT_HOLD` | Seek +120 seconds
`PREV` | Seek -10 seconds
`PREV_HOLD` | Seek -120 seconds
`MENU` | Show progress
`MENU_HOLD` | Mute / unmute audio
`VUP` | Increase audio volume
`VUP_HOLD` | Next chapter
`VDOWN` | Decrease audio volume
`VDOWN_HOLD` | Previous chapter
{{< /table >}}
