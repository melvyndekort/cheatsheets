### Window Management
{{< table >}}
Binding | Description
--- | ---
`Ctrl + Shift + n` | New window
`Alt + F4` | Close window
`Ctrl + Shift + ,` | Reload configuration
{{< /table >}}

### Tab Management
{{< table >}}
Binding | Description
--- | ---
`Ctrl + Shift + t` | New tab
`Ctrl + Shift + w` | Close tab
`Ctrl + Tab` | Next tab
`Ctrl + Shift + Tab` | Previous tab
`Ctrl + Shift + Left` | Previous tab
`Ctrl + Shift + Right` | Next tab
`Ctrl + Page Up` | Previous tab
`Ctrl + Page Down` | Next tab
`Alt + 1-8` | Go to tab 1-8
{{< /table >}}

### Split Management
{{< table >}}
Binding | Description
--- | ---
`Ctrl + Shift + o` | Split right
`Ctrl + Shift + e` | Split down
`Ctrl + Alt + Left` | Focus split left
`Ctrl + Alt + Right` | Focus split right
`Ctrl + Alt + Up` | Focus split top
`Ctrl + Alt + Down` | Focus split bottom
`Ctrl + Shift + Enter` | Toggle split zoom
`Ctrl + Super + Shift + =` | Equalize splits
{{< /table >}}

### Font & Display
{{< table >}}
Binding | Description
--- | ---
`Ctrl + =` or `Ctrl + +` | Increase font size
`Ctrl + -` | Decrease font size
`Ctrl + 0` | Reset font size
{{< /table >}}

### Scrollback
{{< table >}}
Binding | Description
--- | ---
`Ctrl + Shift + Page Up` | Jump to previous prompt
`Ctrl + Shift + Page Down` | Jump to next prompt
`Shift + Page Up` | Scroll page up
`Shift + Page Down` | Scroll page down
{{< /table >}}

### Clipboard
{{< table >}}
Binding | Description
--- | ---
`Ctrl + Shift + c` | Copy to clipboard
`Ctrl + Shift + v` | Paste from clipboard
{{< /table >}}
