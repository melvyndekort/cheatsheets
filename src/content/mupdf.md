### Navigation
{{< table >}}
Binding | Description
--- | ---
`j` / `↓` | Scroll down
`k` / `↑` | Scroll up
`h` / `←` | Scroll left
`l` / `→` | Scroll right
`Page Down` / `Space` | Next page
`Page Up` / `Backspace` | Previous page
`g` | Go to first page
`G` | Go to last page
`123g` | Go to page 123
`.` | Go to next page
`,` | Go to previous page
`t` | Go to top of page
`b` | Go to bottom of page
{{< /table >}}

### Zoom and rotation
{{< table >}}
Binding | Description
--- | ---
`+` / `=` | Zoom in
`-` | Zoom out
`W` | Zoom to fit width
`H` | Zoom to fit height
`Z` | Zoom to fit page
`w` | Shrinkwrap window to fit page
`[` | Rotate counter-clockwise
`]` | Rotate clockwise
{{< /table >}}

### Search
{{< table >}}
Binding | Description
--- | ---
`/` | Search forward
`?` | Search backward
`n` | Find next search result
`N` | Find previous search result
{{< /table >}}

### Display
{{< table >}}
Binding | Description
--- | ---
`i` | Toggle anti-aliasing
`I` | Toggle inverted color mode
`c` | Toggle tinted color mode
`e` | Toggle ICC color management
`f` | Toggle fullscreen
{{< /table >}}

### Misc
{{< table >}}
Binding | Description
--- | ---
`r` | Reload document
`S` | Save document (if permitted)
`q` | Quit
`m` | Mark current page to history
`T` | Go back in history
{{< /table >}}
