### Toolbox
Toolbox is a simple way to get a privileged shell with more tool support and the ability to install more tools
* https://github.com/containers/toolbox/tree/main/doc
```
toolbox list
toolbox create [-i image]
toolbox enter [name]
toolbox rm
```
